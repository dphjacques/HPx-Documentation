fig_type

-fig_type <:txt:Type> <:txt:Option>


Keyword selected_graph


Identifies the type of the plot. Default is type XY.



Type

txt

Text: type of the plot - see tables below

Option

txt

Text: options, which are specific for the plot type - see tables below




Type

Description

XY

XY-plot with the X and X2 variables as independent variables.

Independent variables are defined with -fig_x.

If used with diff, a plot is generated for each variable that is defined as an X variable. 

If used with same, only one plot is defined with the first X variable as the default variables. Alternative X-variables are used if indicated in the -fig_y or -fig_sy identifiers. 

Variables defined with -fig_sx are used if indicated in the -fig_y or -fig_sy identifiers.

Dependent variables are defined with -fig_y or -fig_sy.
