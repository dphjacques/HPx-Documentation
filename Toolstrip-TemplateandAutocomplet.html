Toolstrip - Template and Autocompletion

This tool strip allows the user to set preferences for the insert options and autocompletion.





Insert Options

Insert options define the format of text input into the input area when the user double clicks help items in the help trees.


Following options are available:


Text

The clicked word appears in the input area

Text + Format

The clicked word and the format options (if present) appear in the input area

Text + Default

The clicked word and the default options (if present) appear in the input area

Text + Format + Help

The clicked word and the format options (if present) and an inline help (if present) appear in the input area

Text + Default + Help

The clicked word and the default (if present) and an inline help (if present) appear in the input area


As an example, the different inputs for the -water identifier in the solution datablock:



Text

-water i

Text + Format

-water <:flt:mass>

Text + Default

-water 1

Text + Format + Help

-water <:flt:mass> # Mass of water of the solution. Input: water. Unit: kg. Default: 1

Text + Default + Help

-water 1 # Mass of water of the solution. Input: water. Unit: kg. Default: 1



Autocompletion

Autocompletion defines the list of pop-up suggestions when typing in the input area.


The list can contain:

  • PHREEQC keywords, identifiers, script functions. 
  • Components, species, and phases when 

(i) these are defined in the database or when added via Database Panel (Add Database Information), and

(ii) components are selected   via the button Components in the Database Panel.

  • Global definitions when defined via the Global Definition Panel.


Note that global definitions are always part of the list of suggestions (except in case of None).


Following options are available:


All

Autocompletion list contains all suggestions

None

No autocompletion

Language

Autocompletion list contains all suggestions related to PHREEQC input

Keywords

Autocompletion list contains the PHREEQC keywords

Identifiers

Autocompletion list contains the PHREEQC identifiers

Script Language

Autocompletion list contains the script functions of the MyBasic script language

Chemistry

Autocompletion list contains all chemical information relevant to the selected components

Components

Autocompletion list contains the selected components

Aqueous species

Autocompletion list contains the aqueous species relevant to the selected components

Exchange species 

Autocompletion list contains the exchange species relevant to the selected components

Surface species

Autocompletion list contains the surface species relevant to the selected components

Phases

Autocompletion list contains all phases (minerals and gaseous) relevant to the selected components

Minerals

Autocompletion list contains the minerals relevant to the selected components

Gases

Autocompletion list contains the gases relevant to the selected components
