    Input

    The Input Work Space contains two main parts.


    In HPGeochemistry, following elements are available.



    Toolstrip area

    The toolstrip area shows the current selected input panel and contains:

    • Adding, saving and closing input panels
    • Insert and autocompletion options

    Tab pages

    Different input panels:

    • Database Panel
    • Global Definitions Panel
    • Input file Panels. Some panels are visible by default and cannot be saved - these panels will be saved automatically when the case is simulated. Additional input panels can be added or closed using the toolstrip for adding, saving and closing input panels. The default panels are:
      • Additions to database
      • Geochemistry





    In HYDRUS 5.xx, following elements are available


    Toolstrip area

    The toolstrip area shows the current selected input panel and contains:

    • Adding, saving and closing input panels
    • Insert and autocompletion options

    Tab pages

    Different input panels:

    • Database Panel
    • Reactive Transport
    • Global Variables Panel
    • Input file Panels. Some panels are visible by default and cannot be saved - these panels will be saved automatically when the the buttons OK, PREVIOUS or NEXT is pushed and you return to the HYDRUS graphical user interface.. Additional input panels can be added or closed using the toolstrip for adding, saving and closing input panels. The default panels are:
      • Additions to database
      • Solution Definition
      • Geochemistry
      • Output







    The input panel contains two main parts:


    Help Trees

    At the left hand side, the panal has different tabs for facilitating input.

    Input Area

    At the right hand side is the work area for creating the input for the simulation.
