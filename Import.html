Import

The import form allows for importing thermodynamic data. Two formats are available:

    • Phreeqc format - Thermodynamic data is 


When the data is in XML format, the thermodynamic data can be imported in two ways:

    • Dynamic - the file containing the thermodynamic data is always read when a simulation is loaded. When changes are made to the file between two sessions, the new information is available when the simulation is loaded again;
    • Static - the information in the file is added to the simulation statically in the simulation and saved with the simulation and thus available in the future. However, when the simulation is loaded again, information from the thermodynamic data is not read again and thus, possible changes in the file are not taken into account. The data is imported in the different forms for adding database information and will be visible in the Help Trees. 


A file in PHREEQC format is always read dynamically.


The form is shown below



With the toolstrip at the top, new files can be added or deleted. In addition, the order of the files can be changed.

Information on the data file, its format and the way it is important is given in the cells of the table:


Folder

Select the folder of the file


Default Managed Project

Only available for a managed project. The base folder is the database folder in a managed project. The base folder is used for the relative path to the file.

Database Default Folder

The base folder is the default folder for database groups. The base folder is used for the relative path to the file.

Project Folder

The project folder can be selected as a starting folder to find the file, or as the base folder for the relative path to the file In the former case, an absolute path to the file is defined.

Simulation Folder

The simulation folder can be selected as a starting folder to find the file, or as the base folder for the relative path to the file In the former case, an absolute path to the file is defined.

Last Folder

The last accessed folder is selected as a starting folder to find the file. An absolute path to the file is defined.

Relative path

Indicates either an absolute or a relative path to the file.The option is only enabled for the Folder Project Folder and Simulation Folder. For Default Managed Project and Database Default Folder, only relative paths can be defined and the checkbox is disabled. For Last Folder, only absolute paths can be defined and the checkbox is disabled.

Browse

Opens the File dialogue window for selecting the file with the thermodynamic data.When the Open button is pushed, the absolute or relative path and the file name are visible in the FileName column.

FileName

Absolute or relative path of the file with the additional thermodynamic data.

Import Type

Defines the way the data are imported:

Static - only read once (in the current or future sessions, and imported as "hard" data in the simulation. Only available when the format is XML.

Dynamic - the datafile is read at the start of each session.When the file format is XML, the data is not imported as "hard" data in the simulation.

File Format

Defines the format of the file:

XML - The thermodynamic data is in XML format. This format is obtained by exporting thermodynamic data in XML format from another simulation or project. The format can be imported as Static or Dynamic.

PHREEQC - The thermodynamic data is in PHREEQC format. The format can be imported as Dynamic only.

Use

Indicates if the file is imported in the current simulation.
