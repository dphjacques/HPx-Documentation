Projects

A particular problem is defined as a

simulation in a project in hpgeochemistry

project in a project group in hydrus (see help of HYDRUS for more details)


In hpgeochemistry, a project and its simulations are defined with the Project Manager. The Project Manager gives the user the possibility to create, edit metadata and remove projects. A new project consists of a folder that can be created on any accessible drive. HPGeochemistry has also a default folder in which new projects are created. This default folder can be set via the general settings; the same default folder as in HYDRUS can be used as well. 

The user can specify further details via the Metadata Form. The Metadata Form for a project also allows for:

  1. Setting the default database for this project
  2. Defining the project as a Managed Project


The Project Manager allows also for defining, importing, copying, removing and deleting simulations. Specific information can be added to the Metadata Form for a simulation.


Two methods are available to select and open a simulation:

  1. If the simulation was opened in a previous or the actual session of HPGeochemistry, the simulation can be selected via the Main Form -  Project selection,
  2. via the  Project Manager
