Checking the Simulation Results

Three icons on the main form allow for checking simulation results:

  • The phreeqc.out file with details on reading phreeqc.in and the geochemical calculations in the PHREEQC Out Workspace
  • Figures in the Graph Workspace; Figures can be edited, copied or exported via the Single Graph Panel.
  • Simulation output files in the Datafile Workspace.


In hpgeochemistry,

  • The simulation title in the Main Form is black when there are no warnings or errors, orange when there are warnings, and red when there are errors (and warnings).
  • The listed figures are either defined in the input with the command selected_graph, or constructed via the Datafile Workspace.


In HYDRUS, beside the standard output (in 2D, also the spatial distribution of variables defined with selected_output and USER_PUNCH are shown), additional output is viewed with HPOUTPUT that is activated via the navigation bar Results - Other information HP1 Output or HP2 Output. By default, for the variables defined in  selected_output and USER_PUNCH, following figures are available:

  • Time series plots at the observation points
  • in 1D, profile plots at the selected times
  • in 2D, mesh-line plots (defined via HYDRUS) or cross-section plots
  • Time series of integrated variables (if defined in HPInput).


Additional graphs are defined with selected_graph, to combine different variables in one plot - see Graph Workspace and here.


The different data output files created during the reactive transport simulation can be viewed in the Datafile Workspace.
