REACTIVE_TRANSPORT

reactive_transport defines options for the reactive transport calculations:

  • to indicate which transport properties are changed when geochemical variables evolve.
  • to define the options for multi-threading
  • to indicate if transport simulations have to be performed
  • to indicate if the gas phase is treated as a stagnant phase
