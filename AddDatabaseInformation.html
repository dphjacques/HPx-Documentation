Add Database Information

The button Add Information opens the graphical user interface which allows for adding extra thermodynamic information to the simulation. This information will be available op in the Help Trees. It allows also to export information in XML format (only interpretable by HPx) or PHREEQC format.


       

Selection of the items in the list at the right hand side opens at the left hand side a panel for introducing the new information. Following items are available:

  • Import - To import information in XML or PHREEQC format.
