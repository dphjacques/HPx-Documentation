update

-update <:txt:parameter> <:bool:update>



Keyword reactive_transport


Identifies which transport parameters are updated when geochemical variables changes. The updated values are calculated with scripts defined with calculate_values and the corresponding functionality (see -state).


-state should be repeated for each parameter to be updated.


parameter

txt

Indicate which parameter to be updated

Symbol

parameter

parameter (alternative)

Corresponding functionality (defined by -state in calculate_values )

update

boolean

True/false. Use true of the parameter is updated after each geochemical step. Default: false (constant parameter value).
