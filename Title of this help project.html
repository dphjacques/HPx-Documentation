Introduction

This help file documents the HPx codes and its related graphical user interfaces. HPx (version 3.03.005) contains following codes:



PHREEQC

3.8.2


ORCHESTRA



PYTHON-INTERFACE



HYDRUS1D



 

HYDRUS2D




MT3D_USGS






Graphical user interfaces (version 1.06) are available for 


HYDRUS 5.x



HP-Input

Dialogue window embedded in HYDRUS 5.x graphical user interface to create the input of the geochemical model 

HP-Output

Dialogue window embedded in the HYDRUS 5.x graphical user interface for viewing results of the geochemical model




HPGeochemistry



Software for geochemical calculations based on the PHREEQC version embedded in HPx
