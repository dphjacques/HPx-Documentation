Edit Curves Panel

The Edit Curves Panel allows for modifying individual curves and to inspect curve data and properties. It is accessible via the Edit Graph Tool.



The Edit Curves Panel has

  • A table with the individual curves with some key features
  • A toolstrip for actions on all or multiple selected Curves
  • A toolstrip for actions on or information about a single Curve








Table of individual curves with key information



Label

Alphanumerical value of the label of a curve in the plot legend. The enhanced text mode from gnuplot can be used. 

Visible

The curve is shown or not in the plot. Note that one can activate an automatic selection of visible curves based on the values of the data.



Toolstrip for actions on all or multiple selected Curves
